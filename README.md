# attt-rails

## Launch Command

`make up`

http://localhost:3000

## Setup

```shell
make build
# Install gem
make bundle
# Reset DB and insert seed
make reset
```

*If windows, install something tool so that you can use the make command.

ex)

- Make for Windows: http://gnuwin32.sourceforge.net/packages/make.htm
- scoop: `scoop install make`
